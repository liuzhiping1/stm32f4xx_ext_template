关于本文件夹下子文件夹，这里给大家简单介绍一下，方便大家使用：

1，普通扩展例程。该文件夹下包含了一些普通扩展例程（主要是ALIENTEK配套模块的使用例程），提供寄存器和库函数两个版本，方便大家使用。这些例程一般都配有对应的应用文档(AN)进行解说，各应用文档见：光盘\9，增值资料\2，ALIENTEK应用文档（AN）

2，LWIP扩展例程。包含ALIENTEK提供的LWIP部分例程，教程见《STM32F4 LWIP开发手册.pdf》。

3，EMWIN扩展例程。包含ALIENTEK提供的EMWIN部分例程，教程见《STM32F4 EMWIN开发手册.pdf》。

4，UCOS扩展例程。包含ALIENTEK提供的UCOS部分例程，包含UCOSII和UCOSIII，教程见《STM32F4 UCOS开发手册.pdf》。
