
本实验将实现如下功能：本实验主要讲解如何在BUTTON控件上显示位图，使用GUIBulder制作一个对话框，对话框中有两个BUTTON分别用于控制LED1和蜂鸣器。在BUTTON控件上显示位图用于区分LED1和蜂鸣器的开关状态，打开时显示“ON”，关闭时显示“OFF”

注意：本实验使用了UCOSIII操作系统，本实验使用触摸屏,本实验基于4.3寸800*480分辨率的屏幕，如果使用2.8或者3.5寸屏显示不完全。
